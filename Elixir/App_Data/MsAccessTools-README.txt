ExecutorAssist.MsAccessTools.exe is an application that helps to automatically construct queries for a MS Access database.
How to use it:
1. Put the database file (*.accdb) to the folder with ExecutorAssist.MsAccessTools.exe app.
2. Rename database file to "db.accdb"
3. Run the program.

Note: commands can be shortened:
format-select-all ===> fsa
put-sql-numbers ===> psn

The most commonly used command is "fsa" (format-select-all).
